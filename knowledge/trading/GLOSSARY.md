# Trading Glossary

Quick-reference for terms used in PsiBot trading analysis. Expand acronyms on first use in every response.

## Market Structure

| Term | Definition |
|------|-----------|
| **Bull/Bullish** | Expecting prices to rise |
| **Bear/Bearish** | Expecting prices to fall |
| **Support** | Price level where buying pressure tends to prevent further decline |
| **Resistance** | Price level where selling pressure tends to prevent further rise |
| **Breakout** | Price moving above resistance or below support with conviction |
| **Breakdown** | Price falling below a support level |
| **Consolidation** | Price trading sideways in a range |
| **Gap** | Price jump between sessions with no trading in between |
| **Pullback** | Temporary price decline within an uptrend |
| **Rally** | Sustained price increase |
| **Correction** | 10%+ decline from recent highs |
| **Dead cat bounce** | Brief recovery in a declining market that doesn't hold |

## Technical Analysis (TA)

| Term | Definition |
|------|-----------|
| **RSI** | Relative Strength Index - momentum oscillator (0-100). >70 = overbought, <30 = oversold |
| **MACD** | Moving Average Convergence Divergence - trend/momentum indicator using two EMAs |
| **SMA** | Simple Moving Average - average price over N periods |
| **EMA** | Exponential Moving Average - like SMA but weights recent prices more |
| **BB** | Bollinger Bands - volatility bands around a moving average (upper/middle/lower) |
| **ATR** | Average True Range - measures volatility as average daily price range |
| **VWAP** | Volume Weighted Average Price - average price weighted by volume |
| **Volume profile** | Shows trading volume at each price level |
| **Divergence** | Price and indicator moving in opposite directions (bearish or bullish signal) |
| **Golden cross** | 50-day SMA crosses above 200-day SMA (bullish) |
| **Death cross** | 50-day SMA crosses below 200-day SMA (bearish) |
| **Engulfing** | Candlestick pattern where current candle completely engulfs the previous one |
| **Doji** | Candlestick with nearly equal open and close (indecision) |
| **Hammer** | Candlestick with small body and long lower wick (potential reversal) |

## Options

| Term | Definition |
|------|-----------|
| **Call** | Right to buy at a set price (strike) by expiration |
| **Put** | Right to sell at a set price (strike) by expiration |
| **Strike** | The price at which an option can be exercised |
| **Expiration** | Date when the option contract expires |
| **Premium** | Price paid to buy an option |
| **ITM** | In The Money - option with intrinsic value (call: stock > strike; put: stock < strike) |
| **OTM** | Out of The Money - option with no intrinsic value |
| **ATM** | At The Money - strike price equals current stock price |
| **IV** | Implied Volatility - market's forecast of future price movement priced into options |
| **HV** | Historical Volatility - actual past price movement |
| **PCR** | Put/Call Ratio - ratio of put volume to call volume. >1.0 = more puts (bearish sentiment) |
| **Max pain** | Strike price where most options expire worthless (market makers profit most) |
| **Theta** | Time decay - how much an option loses per day as expiration approaches |
| **Delta** | How much option price moves per $1 move in the stock |
| **Gamma** | Rate of change of delta (acceleration) |
| **Vega** | How much option price changes per 1% change in IV |
| **Spread** | Buying one option and selling another to limit risk/reward |
| **Debit spread** | Spread that costs money to open (pay premium) |
| **Credit spread** | Spread that receives premium when opened |
| **Iron condor** | Combination of bull put spread + bear call spread (profits from low volatility) |
| **Straddle** | Buying both a call and put at same strike (profits from big moves either direction) |
| **Strangle** | Like straddle but with different strikes (cheaper, needs bigger move) |
| **GEX** | Gamma Exposure - net gamma of market makers, influences price magnetism |
| **OI** | Open Interest - total outstanding option contracts |
| **Options flow** | Tracking large/unusual options trades to infer institutional positioning |
| **Smart money** | Institutional traders whose large orders may signal informed positioning |
| **Sweep** | Aggressive order that hits multiple exchanges simultaneously (urgency signal) |
| **Dark pool** | Private exchange where large trades execute without public visibility |

## Fundamental Analysis

| Term | Definition |
|------|-----------|
| **P/E** | Price-to-Earnings ratio - stock price divided by earnings per share |
| **PEG** | P/E-to-Growth ratio - P/E divided by earnings growth rate. <1 may be undervalued |
| **EV/EBITDA** | Enterprise Value / Earnings Before Interest, Taxes, Depreciation, Amortization |
| **FCF** | Free Cash Flow - cash generated after capital expenditures |
| **EPS** | Earnings Per Share |
| **Revenue** | Total income from sales |
| **Margin** | Profit as percentage of revenue (gross, operating, net) |
| **DD** | Due Diligence - thorough research before making an investment |
| **Moat** | Competitive advantage that protects a company's market position |
| **Guidance** | Company's forward-looking earnings/revenue estimates |
| **Earnings surprise** | Actual earnings vs analyst consensus (beat or miss) |
| **Insider activity** | Buying/selling by company executives and directors |

## Macro & Market

| Term | Definition |
|------|-----------|
| **FOMC** | Federal Open Market Committee - sets US interest rates |
| **Fed** | Federal Reserve - US central bank |
| **CPI** | Consumer Price Index - measures inflation |
| **NFP** | Non-Farm Payrolls - monthly US jobs report |
| **GDP** | Gross Domestic Product - total economic output |
| **DXY** | US Dollar Index - dollar strength vs basket of currencies |
| **VIX** | Volatility Index - "fear gauge" measuring expected S&P 500 volatility. >20 = elevated fear |
| **Yield curve** | Plot of bond yields by maturity. Inverted = recession signal |
| **Credit spread (bonds)** | Difference between corporate and Treasury bond yields. Widening = stress |
| **Risk-on** | Market favoring growth/speculative assets |
| **Risk-off** | Market favoring safe havens (bonds, gold, USD) |
| **Sector rotation** | Money flowing from one sector to another based on economic cycle |
| **Flight to safety** | Capital moving to low-risk assets during uncertainty |
| **Hawkish** | Central bank leaning toward higher rates / tighter policy |
| **Dovish** | Central bank leaning toward lower rates / looser policy |
| **Taper** | Reducing asset purchases (less monetary stimulus) |
| **QE** | Quantitative Easing - central bank buying assets to inject money |
| **QT** | Quantitative Tightening - central bank reducing balance sheet |
| **Basis points (bps)** | 1/100th of a percent. 25 bps = 0.25% |

## Energy & Commodities

| Term | Definition |
|------|-----------|
| **Crack spread** | Profit margin between crude oil and refined products (gasoline, diesel) |
| **Refiner** | Company that processes crude oil into usable products (VLO, PSX, MPC) |
| **WTI** | West Texas Intermediate - US crude oil benchmark |
| **Brent** | Brent Crude - international oil benchmark |
| **Contango** | Futures price > spot price (normal, storage cost) |
| **Backwardation** | Spot price > futures price (supply squeeze) |
| **OPEC** | Organization of Petroleum Exporting Countries |

## Trading Strategy

| Term | Definition |
|------|-----------|
| **Sharpe ratio** | Risk-adjusted return. >1 = good, >2 = very good |
| **Drawdown** | Peak-to-trough decline during a specific period |
| **Max drawdown** | Largest historical drawdown (worst-case loss) |
| **Profit factor** | Gross profits / gross losses. >1.5 = good |
| **Win rate** | Percentage of trades that are profitable |
| **R:R** | Risk-to-Reward ratio. 1:3 means risking $1 to make $3 |
| **Position sizing** | How much capital to allocate to each trade |
| **Stop loss** | Price level where you exit a losing trade |
| **Take profit** | Price level where you exit a winning trade |
| **Mean reversion** | Strategy betting prices return to average |
| **Trend following** | Strategy betting current trend continues |
| **Momentum** | Strategy buying recent winners / selling recent losers |
| **Backtest** | Testing a strategy on historical data |
| **Paper trading** | Simulated trading without real money |
| **Regime** | Current market conditions (trending, ranging, volatile, calm) |
| **Alpha** | Returns above a benchmark (outperformance) |
| **Beta** | How much a stock moves relative to the market |
| **Hedge** | Position that offsets risk in another position |
| **Binary event** | Upcoming event with two discrete outcomes (earnings, FDA decision, geopolitics) |

## Signal Scores

| Term | Definition |
|------|-----------|
| **Signal score** | PsiBot's composite score (0-100) combining TA, sentiment, and options data |
| **Bullish signal** | Score >60, suggesting upward bias |
| **Bearish signal** | Score <40, suggesting downward bias |
| **Conflicting signal** | Different indicators disagreeing (e.g., TA bullish but options bearish) |
| **Confidence** | How strongly a signal is supported across multiple data sources |
